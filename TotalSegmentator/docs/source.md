# Source

TotalSegmentator is an open-source segmentation tool and dataset project from
University Hospital Basel.

Official project:

```text
https://github.com/wasserth/TotalSegmentator
```

The project README links to these dataset records:

- CT dataset: `https://zenodo.org/records/10047292`
- MRI dataset: `https://zenodo.org/records/14710732`

The CT dataset record describes 1228 CT images with 117 segmented anatomical
structures. The MRI dataset record describes 616 MR images with 50 segmented
anatomical regions.

Licenses differ by dataset:

- CT v2.0.1: Creative Commons Attribution 4.0 International.
- MRI v2.0.0: Creative Commons Attribution Non Commercial Share Alike 2.0 Generic.
