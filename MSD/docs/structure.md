# Structure

```text
MSD/
  data/raw/archives/
  data/raw/extracted/
  data/processed/
  data/manifests/
  data/splits/
  scripts/
  src/msd/
  docs/
  logs/
  cache/
```

`data/raw/archives/` stores original downloads. `data/raw/extracted/` stores
unpacked task directories such as `Task01_BrainTumour`.
